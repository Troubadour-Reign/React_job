# React Job App

A simple job listing web app built using React, JSON Server, and Tailwind CSS.

## Features

- List of job openings.
- Add, update, and delete job listings.
- Responsive design using Tailwind CSS.

## Technologies Used

- React
- JSON Server
- Tailwind CSS
